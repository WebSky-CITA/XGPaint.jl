    XGPaint.get_cosmology — Method.

    Construct a background cosmology.

    This function duplicates the cosmology() function in Cosmology.jl, but with typing. The type of the cosmology will the type of h and OmegaM. This is primarily for keeping the code entirely in Float32 or Float64.
