    XGPaint.hod_sehgal! — Method.

    Populate halos with radio sources according to the HOD in Sehgal et al. 2009.

    The optional rng parameter provides an array of random number generators, one for each thread. If none are specified, this function will call XGPaint.get_thread_RNG() to create such an array, which takes about 1 second.
